# Booking package carousel images

Place images here for the booking page package carousels. The app expects these filenames:

**Listing packages** (3 images each)
- `essential-1.jpg` … `essential-3.jpg`
- `premium-1.jpg` … `premium-3.jpg`
- `luxury-1.jpg` … `luxury-3.jpg`

**Personal branding packages**
- `growth-1.jpg` … `growth-3.jpg` (3 images)
- `accelerator-1.jpg` … `accelerator-3.jpg` (3 images)
- `tailored-1.jpg`, `tailored-2.jpg` (2 images)

You can use `.jpg`, `.jpeg`, or `.png`; update `database/init.sql` and `database/migrate-booking-package-images.sql` if you use different extensions.
